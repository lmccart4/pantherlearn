Hiring Filter Rule Cards
Filter Rule #1
“Reject any resume with an employment gap of 6 months or more.”
Mark an X on any resume with a visible 6+ month gap.
Filter Rule #2
“Accept only candidates from top-50 universities.”
Top-50 in this dataset: Columbia, NYU, Princeton, Penn. Mark a check on qualifying resumes; X on all others.
Filter Rule #3
“Reject any resume from a zip code with more than 20% unemployment.”
High-unemployment zips: 07202, 07206, 07520, 07032.
Filter Rule #4
“Accept only candidates under age 35.”
Circle the age on each resume. Mark X on anyone 35 or older.
Filter Rule #5
“Reject any resume without 3 or more years of continuous employment.”
Look at the longest unbroken stretch. Less than 3 years = X.
Filter Rule #6
“Accept only candidates whose skills include Excel AND one of: Python, SQL, or Tableau.”
Both conditions must be true. Missing either = X.
