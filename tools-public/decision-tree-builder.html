Decision Tree Builder

Build, visualize, and walk through decision trees

How to Use

  1. Start with a template or build from scratch using the "New Root" button
  2. Click a node to select it, then add child branches (Yes/No, options, outcomes)
  3. Drag nodes to rearrange the layout
  4. Use "Walk Through" to interactively follow the decision tree step by step
  5. Export your tree as text to share or save
